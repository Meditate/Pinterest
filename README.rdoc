== README

Hello there,

This is simplified pinterest-type servise, which I've made with Meditate.

Gems we used:
  -haml
  -devise
  -simple_form
  -bootstrap
  -carrierwave
  -act_as_votable
  
Functionality:
  -user authentication
  -creating posts with image, title and content
  -like/dislike function
  -random post function
  -comments
  -search form
